# QLS-MiCM Training Hub 💻📊🧬⚕️🔬

**Welcome to the QLS-MiCM GitHub page!**

This repository serves as our central training hub, hosting materials, resources, and code from the various QLS-MiCM workshops and training sessions.

## Learning Pathways
New here? Our [**Learning Pathways**](https://github.com/QLS-MiCM/.github/blob/main/LEARNING_PATHWAYS.md) guide groups our workshops into structured tracks — from programming basics through machine learning, RNA-seq, genetics, and more — so you can follow a clear sequence from foundations to advanced topics.

## Instructors
Please refer to the [Workshop Repo Template](https://github.com/QLS-MiCM/Workshop_Template) for detailed setup instructions, contribution guidelines, and repository structure for updating materials or submitting new workshops. 

## Contact
If you have questions, feedback, or requests related to a specific workshop, please reach out by **submitting an issue** on that workshop’s repository.
You can also contact us directly via email at workshop-micm@mcgill.ca!
